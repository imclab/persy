Changelog¶

You can find the latest commit history on github.

the raw beauty¶

0.9.1 - Sun, 14 May¶

  • documentation updates
  • added support for GIT_WORK_TREE and GIT_DIR in the configuration
  • added support for multiple configurations
  • experimental support for fancy remote git servers (like gitoris)
  • bug fixes

0.9ubuntu1 - Sun, 02 May¶

  • Switch to dpkg-source 3.0 (quilt) format

0.9 - 18 Feb¶

  • new and improved documentation
  • new build and installation system
  • new logos for linux mint and fedora
  • new ability to easy start persy at login
  • new and easy autorun at login options
  • added support for different remote server ports
  • added list of submodules (files in submodules are not tracked in persy)
  • possibility to install persy on every linux distribution (make install!)
  • language updates
  • settings gui updates
  • some bugfixes and cleanups

0.8.1 - 05 Feb¶

  • added headless mode
  • added username for serverauth
  • some bugfixes and cleanups

0.8 - 27 Jan¶

  • rewrote the settings gui
  • added environment tests

0.7.2.1 - 19 Jan¶

  • moved the doc building code out of debian/rules and in its own script
  • fingers crossed it will NOW build on launchpad!

0.7.2 - 18 Jan¶

  • fixed wrong glade file
  • using pandoc for most of the documentation
  • added help functionality and help files

0.7.1 - 16 Jan¶

  • fixing missing glade dependency
  • added autoshare ability to the configuration file
  • some bugfixes

0.7 - 10 Jan¶

  • new settings gui
  • removed initLocal command
  • readme updated
  • added manpage autoupdate
  • lots of bugfixes

0.6.2 - 21 Dec¶

  • added a last remote sync when closing persy
  • updated package
  • readme updated

0.6.1.3 - 03 Dec¶

  • fixed a problem with a non-existant configuration file
  • added an experimental gui frame (will add more in the future)

0.6.1.2 - 26 Nov¶

  • fixed a show-stopper bug, thanks again emorrp1

0.6.1.1 - 26 Nov¶

  • fixed a show-stopper bug, thanks emorrp1

0.6.1 - 23 Nov¶

  • removed an warning when changing files in a git submodule

0.6 - 23 No¶

  • The “almost there” release
  • reorganized every single line of code
  • lots of bugfixes

0.5 - 27 Oct¶

  • The “media whore” release
  • localization started
  • readme improved
  • updated configfile with more default values
  • updated configfile with some explanations
  • replaced a deprecated python-apt function
  • set icon svg size to 128x128px (and fixing a gtk warning with this)
  • implemented delayed error reporting (finaly!)

0.4.3 - 18 Oct¶

  • added a config option for a prefered gui browser
  • added active remote sync
  • lots of bugfixes

0.4.2 - 1 Sep¶

  • lots of bugfixes

0.4.1 - 25 Aug¶

  • added support for short forms of added directories

0.4 - 13 Aug¶

  • fortune support
  • new icons for unsynced and untracked
  • lots of bugfixes

0.3.6 - 11 Aug¶

  • added svg icons for all the stati
  • added secret cli option to set environment variables in the parent shell
  • improved statusicon tooltip
  • improved display of logs
  • added support for qgit (gitk is still prefered)
  • lots of bugfixes

0.3.5.1 - 06 Aug¶

  • added svg icon

0.3.5 - 06 Aug¶

  • fixed path to VERSION file
  • make the code more robust
  • added GPL in the about dialog
  • added optimize function
  • added new icons for untracked and unsynced files
  • added new svg icons
  • created a markdown README file
  • lots of bugfixes
  • added untracked and unsynced status (experimental)

0.3.4 - 27 Jul¶

  • added bash completion
  • code cleanups (logging, gtk, config, threads)
  • new VERSION file introduced
  • added exclusion of files by pattern
  • added exclusion of files bigger than x kb
  • removed git-gc after a commit for performance

0.3.3 - 22 Jul¶

  • Bugfix release:
  • git logging
  • config warnings
  • code cleaning
  • inotify KeyError
  • error handling in pug.init

0.3.2 - 21 Jul¶

  • added desktop entry + icon
  • added notifications on critical error
  • fixed typo that was breaking any config attempts – thanks to Philip Morrell

0.3.1 - 21 Jul¶

  • fixed absolute path problem

0.3 - 21 Jul¶

  • fist implementation of a tray icon
  • much better error handling
  • lots of bugfixes

0.2.1 - 14 Jul¶

  • small bugfixes which prevented 0.2 to work

0.2 - 14 Jul¶

  • introduced the git abstraction PuG
  • lots of changes
  • lots of cleanups
  • lots of bugfixes
  • yay

0.1 - 07 Jul¶

  • Initial release
