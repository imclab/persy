Developer Documentation
=================================
This is an overview what you might want to know if you want to hack and contribute to persy.

CLI
--------------------------------
A development Option is --setenv. This sets the environment variables (GIT_DIR, GIT_WORK_TREE) of the parent shell in a way that all git commands execute in the parent shell are operating on the persy git repository. Take a look at the help message when using this option.

Contributing
---------------------------------
If you want to contribute in anyway, please go to github (http://github.com/kinkerl/persy) or launchpad (http://github.com/kinkerl/persy).

Dependencies
--------------------------------

 *   git-core
 *   gitk or qgit - graphical git browser
 *   xterm
 *   librsvg2-common
 *   python-gtk2
 *   python-glade2
 *   python-notify
 *   python-pyinotify - to get efficient information about filesystem changes
 *   python-paramiko - ssh library to initialize a remote server

For some actions in the Makefile you also need

 *   python-sphinx
 *   pandoc
